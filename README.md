# sancakkit
A simple OSINT tool

# Features
**Instagram Ip Query**  
**Whois Query**  
**Username Search**  
**Instagram OSINT**  

# INSTALLATION
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/charliecpln/sancakkit.git
   cd sancakkit
   ```

2. **Run the Application**:
   Start the application with:
   ```bash
   sudo python sancakkit.py
   ```
